# todolist

## Overview

A todo list app

## Installation

Make sure you have `node` installed on your computer

1. Clone this repo

```bash
git clone https://github.com/RonenTGreat/todolist.git
```

2. Navigate to `directory`

```bash
cd todolist
```

3. Install `dependencies`

```bash
npm i
```

4. Run app

```bash
node app.js
```
